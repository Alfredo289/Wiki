Open-source project by shann-holmberg that implements andrej-karpathy's llm-wiki-pattern as a ready-to-clone obsidian vault. Agent-agnostic - works with Claude Code, OpenClaw, Hermes, Codex, or custom setups.

Core operations: INGEST (process raw sources), QUERY (research and file answers), EXPLORE (expand topics), LINT (health check).

Uses qmd for local search at scale.
