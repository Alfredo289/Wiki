Open-source macOS tool for system-level key remapping and behavior modification. Used by Brett Terpstra primarily for Hyper key setup, which unlocks additional keyboard shortcut possibilities. Website: karabiner-elements.pqrs.org.

Part of the keyboard modification layer in a keyboard-driven-workflow, alongside BetterTouchTool, Keyboard Maestro, and DefaultKeyBindings.dict.
