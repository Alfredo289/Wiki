Open-source persistent memory system for Claude Code by Alex Newman (@thedotmack). Captures observations during tool interactions, compresses them into semantic summaries, and makes them available across sessions.

Architecture: lifecycle hooks → SQLite (FTS5) + Chroma vector DB → 3-layer progressive disclosure (search → timeline → full details). Claims ~10x token savings vs loading all context.

Part of the context-compression landscape. Complements Caveman Compress (which compresses files directly instead of adding a retrieval layer).

Supports Claude Code, Gemini CLI, and OpenClaw. Licensed AGPL-3.0.
