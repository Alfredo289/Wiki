Source: Blog post by Brett Terpstra, 2026-03-27 Context: Terpstra is a developer and writer focused on macOS productivity tooling.

Summary

A curated overview of 20 macOS utilities organized around the principle that the keyboard is the most powerful input tool on a Mac. Terpstra uses all of these daily and describes how they complement each other.

Categories and Tools

Keyboard Navigation

  • Superkey - searches text on screen, simulates mouse click on matching text. Navigate without leaving keyboard.
  • Wooshy - alternative to Superkey for navigating all UI elements via keyboard.
  • KindaVim - brings Vim-like key bindings (h/j/k/l/u/d/a/i) to any macOS app. Not perfect but great for basic Vim navigation.
  • Paletro - Shift+Cmd+P in any app for fuzzy menu item search. Used daily.
  • Scrolla - keyboard-driven scrolling through documents.

Keyboard Modifications

  • BetterTouchTool - trackpad gesture and keyboard shortcut manager. Creates personalized commands.
  • Karabiner-Elements - system-level key remapping. Terpstra uses it for Hyper key setup.
  • DefaultKeyBindings.dict - macOS built-in key mapping file for custom keyboard shortcuts.
  • Keyboard Maestro - automation engine for repetitive tasks. Combined with key bindings for streamlined text editing.

Launchers

  • LaunchBar - Terpstra’s preferred launcher after trying all alternatives. Features: clipboard history across reboots, Instant Send (double-tap Option), file actions. Combined with KindaVim and Superkey, creates a powerful navigation system.
  • LeaderKey - static single-key sequences for launching frequently used apps and URLs. Complements LaunchBar.

Text Expansion

  • TextExpander - longtime tool with snippet search, reminders, shell script execution. Expansion after whitespace feature gives it an edge.
  • TextBlaze - alternative text expander, but Terpstra finds using both confusing.
  • Cotypist - current favorite. AI-powered automatic text completion in any app. Predicts inputs without pre-created snippets. Works in Cursor too.

Window Management

  • Moom - move and resize windows via keyboard. Grid mode for drawing positions on the fly.

Web Browsing

  • Vimium - Vim-style navigation for browsers. Press F to tag visible links with key sequences.
  • Kagi - search engine with full keyboard navigation and AI slop filtering.

Key Insights

  • The tools form an ecosystem where each fills a gap the others leave. The power is in the combination, not individual tools.
  • Terpstra mentions an unpublished script based on NiftyMenu that scans any app for available shortcuts and generates Dash cheatsheets.
  • The LaunchBar + KindaVim + Superkey combination is highlighted as particularly powerful for mouse-free navigation.

Counter-arguments

  • 20 utilities is a heavy investment in configuration and muscle memory
  • Many tools overlap in functionality (e.g., BetterTouchTool can do some of what Keyboard Maestro and Karabiner do)
  • Using both TextExpander and TextBlaze simultaneously suggests even the author hasn’t fully rationalized the stack
  • Some tools (Cotypist) rely on AI prediction which may not suit all writing styles

Data gaps

  • No discussion of how long it took to build this workflow or the learning curve
  • No mention of conflicts between tools (key binding collisions, etc.)
  • Missing cost breakdown - several are paid apps
  • No comparison with built-in macOS keyboard features that might replace some of these tools
