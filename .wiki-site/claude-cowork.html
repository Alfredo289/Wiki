Anthropic’s desktop productivity tool, positioned between simple chat and terminal-based Claude Code. Available in the Claude desktop app as the middle tab (Chat | Cowork | Code).

Designed for non-engineers who want AI to execute tasks, not just advise. Key features:

  • Folder-based projects with shared memory across conversations (reinforces folder-as-agent)
  • One-click connectors to Gmail, Slack, Notion, Google Calendar with granular permissions
  • Skills - reusable instructions for specific tasks (writing style, advisory boards, research)
  • Scheduled tasks - recurring automations (e.g., morning debrief at 7:30 a.m.)
  • Sub-agents - multiple AI personas for multi-perspective review

The “brain file” pattern (a preferences document Claude reads every session) is functionally identical to an agentic-constitution.

Counter-arguments

  • May be a UI wrapper that adds friction vs direct Claude Code usage
  • Connector quality determines the ceiling of the tool

Data gaps

  • Single tutorial source - no comparative reviews or benchmarks
  • Cost/token implications of scheduled tasks and sub-agents unclear
