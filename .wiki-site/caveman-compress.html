Open-source Claude Code skill by Julius Brussee that reduces token consumption by compressing project memory files (CLAUDE.md, todos, preferences) while preserving semantic content.

Achieves ~46% average token reduction. Uses a pipeline: file detection → Claude-powered compression → validation → targeted repairs. Preserves code blocks, URLs, file paths, and structural elements.

Part of the context-compression landscape. Complements Claude-Mem (which uses progressive disclosure instead of file compression).

Command: /caveman:compress <filepath>. Requires Python 3.10+.
