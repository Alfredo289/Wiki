macOS application launcher and productivity tool by Objective Development. Features: clipboard history across reboots, Instant Send (double-tap Option key), file actions, quick calculations. Brett Terpstra's preferred launcher. Website: obdev.at/launchbar.

Combined with KindaVim for Finder navigation and Superkey for screen text selection, forms a powerful mouse-free navigation system.
